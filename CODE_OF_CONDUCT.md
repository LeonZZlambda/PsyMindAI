# Code of Conduct

This project follows a standard Code of Conduct for maintainers and contributors. Respect, inclusion, and professionalism are required.

Short version:

- Be kind and respectful.
- Do not use offensive language, harassment, discriminatory remarks, or personal attacks.
- Contributions from people with diverse backgrounds and experiences are welcome.

How to report:

If you experience or observe inappropriate behavior, report it privately to the project maintainers using GitHub Security Advisories or by emailing the maintainers listed in the repository (use a private issue or advisory when applicable).

Consequences:

Behavior that violates this code may result in removal of comments, blocking of accounts, or removal of contributions at the maintainers' discretion.

This document is a short summary — formal procedures and policies used by the project apply for serious cases.
