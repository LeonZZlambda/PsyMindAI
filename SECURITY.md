# Política de Segurança

Se você encontrou uma vulnerabilidade de segurança, agradecemos a divulgação responsável.

Como reportar:

1. Prefira o recurso de Security Advisories do GitHub para envio privado.
2. Se não puder usar o GitHub Security Advisory, abra um email para os mantenedores (ou use o canal indicado no repositório), incluindo: descrição, passos para reproduzir, impacto e PoC mínimas.

Tempo de resposta e boas práticas:

- Confirmaremos o recebimento em até 72 horas.
- Não publique detalhes até que um patch ou mitigação esteja disponível.

Divulgação responsável:

Trabalharemos com você para coordenar a correção antes da divulgação pública. Se desejar crédito, informe durante o contato.

Nota: este projeto é mantido por colaboradores voluntários; priorizamos correções segundo gravidade e impacto.
